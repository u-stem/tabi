Here is a summary of updates since launch.

## New Features

### Friends

You can now send and accept friend requests. Being friends makes it easier to invite members to trips.

### Candidate Reactions

You can now react to candidate spots with "Like" or "Hmm." You can also sort candidates by popularity.

### CSV / Excel Export

Export your itinerary in CSV or Excel format. CSV supports customizing the delimiter (comma / tab), BOM, and line endings (CRLF / LF).

### QR Code Sharing

A QR code dialog is now displayed when generating share links. You can share via QR code in addition to copying the URL.

### Multi-Day Labels

Labels such as "Staying" and "Continuing" are now automatically displayed for accommodations and multi-day activities.

### FAQ Page

A [FAQ page](/faq) has been added.

## Improvements

- Redesigned the print / PDF page to a table format for better readability
- Improved the shared link page layout with better arrangement of time, name, and details
- Made pattern action buttons easier to find and tap
- Improved mobile usability for a more app-like experience
- Optimized skeleton loading to reduce screen flickering
- Tapping an address now opens it in Google Maps
- Strengthened security headers

Please send us your feedback from the Settings page.
