## New Features

### Bookmark Lists

You can now organize favorite spots, restaurants, and reference links into lists. Great for gathering candidates before planning a trip.

- Access from "Bookmarks" in the navigation
- Set name and visibility (private / friends only / public) per list
- Each bookmark supports a name, memo, and up to 5 URLs
- Bulk delete and duplicate in selection mode

### Bookmark-Trip Integration

A "Bookmarks" tab has been added to the trip detail page. Add bookmarks as candidates or save schedules/candidates to bookmarks.

- Bookmarks tab: Select a list and bulk-add to candidates
- "..." menu on schedules/candidates: "Save to Bookmarks"
- Bulk operations in selection mode are also supported

### Public Profile

When a bookmark list's visibility is set to "Public," anyone can view it from your profile page.

- Public list bookmarks are visible to anyone
- Copy and share your profile URL from the Settings page

Please send us your feedback from the Settings page.
