## New Features

### Groups

Create groups like "Family," "Coworkers," or "Club" to organize your friends. When adding members to a trip, you can bulk-invite from a group.

- Create, edit, and delete groups from the "Friends" page
- Add friends or users directly by user ID
- "From Group" tab added to the member add dialog
- Select a group to bulk-add members who aren't already in the trip

### Friend Requests on Home Screen

Friend request notifications are now displayed at the top of the home screen. Hidden when there are no pending requests.

## Improvements

- Redesigned the Friends page to combine friend list, groups, and requests on one page
- Added scrollable display for long friend and group lists
- Already-added users are now hidden in the member add dialog for easier selection
