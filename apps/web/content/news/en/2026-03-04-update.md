## New Features

### QR Code Friend Requests

Tap the "QR Code" button on the profile page to display your QR code. When someone scans it, a confirmation page opens where they can send a friend request. No need to share your user ID manually.

## Improvements

### Profile Page Layout

The profile edit button and QR code button are now displayed side by side.
