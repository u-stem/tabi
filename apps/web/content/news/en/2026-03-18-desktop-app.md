## Desktop App

A desktop app for macOS and Windows has been released.

Download the installer from "Download Desktop App" in the "Other" tab of the Settings page.

### Features

- Native OS window management (Dock / Taskbar icon)
- Auto-update (you'll be notified when a new version is available)
- Full feature parity with the web version

### Installation Note

The app is not currently code-signed, so you may see a warning on first launch.

- **macOS**: Right-click (or Control+click) > "Open"
- **Windows**: "More info" > "Run anyway"

Please send us your feedback from the Settings page.
