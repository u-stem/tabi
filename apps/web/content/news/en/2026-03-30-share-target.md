## New Features

### Share from Browser

You can now add restaurant and sightseeing pages you find in your phone's browser directly to sugara via the share menu.

- Select sugara from your browser's "Share" button to add a URL as a candidate
- The page title is automatically set as the candidate name
- If you have only one active trip, the trip selection is skipped for instant addition
- Available when sugara is installed as a PWA

### Auto-fill Title from URL

When you enter a URL in the candidate or schedule form, the page title is automatically filled into the name field.

- Auto-fill only occurs when the name field is empty
- If a name has already been entered via Google Maps search, it will not be overwritten

Please send us your feedback from the Settings page.
