# 日本ルーレットツール 設計ドキュメント

## 概要

`/my` ページに「ツール」セクションを追加し、ルーレットページ (`/tools/roulette`) へのリンクカードを置く。旅行先やアクティビティをサッと決めたいユーザー向けのランダム選択ツール。

## 3モード

1. **都道府県モード** - 47都道府県からランダム。地域フィルタ（北海道・東北、関東、中部、近畿、中国・四国、九州・沖縄）で絞り込み可
2. **カスタムモード** - ユーザーが選択肢をテキスト入力。追加/削除してルーレット
3. **ブックマークモード** - 自分のブックマークリストを選択し、その中のアイテムからランダム

## UI

- タブで3モード切替
- 大きめの結果表示エリア（テキストシャッフル → 減速 → 確定のアニメーション）
- 「回す」ボタン1つ
- 都道府県モード: 地域チップで複数選択フィルタ
- カスタムモード: テキスト入力フィールド + 追加ボタン、選択肢一覧（削除可）
- ブックマークモード: リスト選択ドロップダウン → リスト内のアイテムが候補に

## データ

- 都道府県データはフロントエンドに定数として持つ（APIリクエスト不要）
- カスタムモードの選択肢はローカルステートのみ（DB保存なし）
- ブックマークモードは既存の `GET /api/bookmark-lists` と `GET /api/bookmark-lists/:id` を利用

## スコープ外

- 結果の保存・履歴
- 共有機能
- DB変更なし（フロントエンドのみの機能）
